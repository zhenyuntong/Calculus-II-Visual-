u-Substitution

LEARN  u-Substitution — renaming the variable to simplify the integral ▶
The Idea

If you spot a function u = g(x) whose derivative g′(x) also appears in the integrand, you can rename everything in terms of u. It works because the chain rule says a tiny u-change is du = g′(x)dx, so the thin x-strip and the matching u-strip represent the same signed area.

Formula Decoded
Set u = g(x) ← the "inner" function
Then du = g′(x)dx ← differentials transform
Bounds: x=a → u=g(a), x=b → u=g(b)
Original function: h(x)=f(g(x))g′(x)
The two colored areas are equal because each strip is remeasured, not changed
Try This

Load "∫sin³(x)cos(x)dx". u = sin(x), f(u) = u³. The derivative cos(x) is already there.
Key Insight

The two plots always have equal shaded areas — substitution is a change of coordinates, not a change of problem.

# Calculus-II-Visual-
<img width="1280" height="720" alt="Colored 3D surface" src="https://github.com/user-attachments/assets/7d31ffbc-8c89-4eec-969c-82ab5890f90b" />

An interactive learning app that explains calculus through movable 2D and 3D graphs, guided tutorials, and step-by-step equation solving for integration, multivariable calculus, optimization, and differential equations. It is free and open-source under the MIT License, so anyone may use, modify, and share it while keeping the original copyright and license notice.

## What is included

- Riemann sums
- u-substitution
- Integration by parts
- Partial derivatives and gradients
- Lagrange multipliers
- Linear ODEs
- Leibniz integral rule

Every lesson has an interactive graph, a guided tutorial, and step-by-step equations.

## Open on a Mac

1. Download this folder.
2. Double-click **Open Calculus Visualizer.command**.
3. Keep the Terminal window open while using the app.

If macOS blocks it: Control-click the file, click **Open**, then click **Open** again.

## Open on Windows

1. Download and unzip this folder.
2. Double-click **Open Calculus Visualizer.bat**.
3. Keep the Command Prompt window open while using the app.

If Windows shows a warning, click **More info**, then **Run anyway**.
